How to use these pins

1. Right-click on any pin below → Inspect

2. In DevTools, find the <div class="pin"> element (the one you right-clicked).

3. Right-click on that element in the inspector → Capture node screenshot

4. A PNG (1000×1500) downloads automatically — upload it to Pinterest as a pin.

Works in: Chrome, Edge, Brave, Opera, Arc. In Firefox: use the “Screenshot Node” option in the inspector instead.
